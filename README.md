# web

## How to configure
* Set server address on
    * resources/views/includes/config.jade
    * resources/assets/js/app.js
* Create `config.php` file using example on `config` directory

## How to compile
type `gulp`

## How to test locally
type `gulp serve`

**Note:** all actions must be executed in the `WEBSITE` directory
